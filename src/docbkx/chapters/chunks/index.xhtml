Using security groups with cURL

These sections walk you through creating, using, and modifying security groups and rules by using cURL.

Start here: “Create a security group (cURL)”

Create a security group (cURL)

The following steps shows you how to create a security group.

Procedure 1. To create a security group

  1. Issue the following cURL command, substituting your own values for the ones shown: 

                            $ curl -i https://dfw.networks.api.rackspacecloud.com/v2.0/security_groups \
       -X 'POST' \
       -H "X-Auth-Token: $token" \
       -H "X-User-Agent: python-novaclient" \
       -H "Accept: application/json" \
       -d {"security_group":{"name":"1-new-tcp","description":"security group for webservers"}} | python -m json.tool
    Note

    Include the -i option in the cURL command to show the response header.

    The operation returns the response body, as shown in the following example: 

    {
    "security_group": {
        "description": "security group for webservers", 
        "id": "fd269d7a-0909-488f-b5fc-5606c8aa489f", 
        "name": "1-new-tcp", 
        "security_group_rules": [], 
        "tenant_id": "5831008"
    }
}

  2. Make note of the id value from the output. You will use this value in later activities. In this example, the ID is fd269d7a-0909-488f-b5fc-5606c8aa489f, but use the ID from your response.

Delete a network (cURL)

To delete a network, specify the network ID.

Procedure 2. To delete a specified network

  • Issue the following cURL command, substituting your own values for the ones shown: 

                            $ curl -i https://dfw.networks.api.rackspacecloud.com/v2.0/networks/29f52c7e-6efd-4335-a14a-db77d32a2555 \
       -X 'DELETE' \
       -H "X-Auth-Token: $token" \
       -H "X-Auth-Project-Id: $account" \
       -H "Accept: application/json"
    Note

    Include the -i option in the cURL command to show the response header. Omit the | python -m json.tool clause from the command because the API operation does not return a JSON response.

    Argument:

    • id. The ID of the subnet that you want to delete. In this example, the subnet ID is 23e3059e-4f39-4f7f-8cf2-c326e5de6c37.

    Output:

    Successful deletion returns an 202 Accepted response code:

    HTTP/1.1 202 Accepted
Date: Thu, 04 Oct 2012 16:23:30 GMT
Content-Length: 58
Content-Type: text/html;charset=UTF-8
Server: Jetty(8.0.y.z-SNAPSHOT)

202 Accepted

The request is accepted for processing.
